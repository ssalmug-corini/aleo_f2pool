# 사용법

------------------------------------------------------------------
필수

```
apt update
apt install git nvidia-cuda-toolkit -y
```

셋업

```
git clone https://github.com/ssalmug-corini/aleo_f2pool
cd aleo_f2
```

실행

```
chmod +x aleo-prover-cuda && chmod +x aleo.sh && ./aleo.sh
```
실행 후 f2pool 유저네임 입력

로그확인
```
cd ./aleo_f2 & cat prover.log
```
